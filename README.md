# Breakout - Pygame

Juego Breakout desarrollado en Python con Pygame para proyecto universitario.

## Ejecutar en Windows

```powershell
pip install -r requirements.txt
python main.py
```

## Compilar .exe

```powershell
.\build.ps1
```

El ejecutable queda en `dist\Breakout.exe`.

## Android

- Guia Pydroid: `ANDROID.md`
- Compilar APK: `BUILD_APK.md`
- GitHub Actions: pestana Actions → **Build Android APK**

## Controles

| Accion | Teclado / PC | Movil |
|--------|--------------|-------|
| Iniciar / lanzar | Espacio / clic | Tocar |
| Mover barra | Flechas, A-D, mouse | Arrastrar dedo |
| Pausa | P | - |

## Mejoras (power-ups)

- **3x** Multibola
- **W** Barra ancha
- **+1** Vida extra
- **S** Pelota lenta
- **F** Paleta rapida
